Tools and Services for Lexicon Visualisation

LiveLanguage Lexicon Viewer

A simple, multi-platform tool for viewing the contents of monolingual lexicons. It is similar to the well-known WordNet View user interface. It can directly load any LMF/XML-based lexicon downloaded from the LiveLanguage catalogue.

Type: offline tool.

Status: released (beta).

Available at: .

Local Knowledge Core (LKC) Website

Visualise on the web the contents of a local lexical database (LKC), providing browsing and visualisation facilities, as well as project information.

Example LKC websites:

  • IndoUKC (Indian languages): http://indo.ukc.datascientia.eu
  • Arabic LKC (Arabic languages and dialects): http://arabic.ukc.datascientia.eu

Type: website.

Status: released.

Available at: contact us if you wish to install a local website instance.
