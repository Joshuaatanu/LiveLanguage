Tools and Services for Lexicon Development

Local Knowledge Core (LKC) Database Management System

An open-source, simple-to-use, Postgres-based database system for storing, developing, and serving lexical data. It is a fundamental component of any local language development or visualisation project. It can import, export, modify, and serve multiple interconnected lexicons.

Type: offline application.

Status: released (alpha).

Available at: .

LiveLanguage Lexicon Viewer

A simple, multi-platform tool for viewing the contents of monolingual lexicons. It is similar to the well-known WordNet View user interface. It can directly load any LMF/XML-based lexicon downloaded from the LiveLanguage catalogue.

Type: offline application.

Status: released (beta).

Available at: .
